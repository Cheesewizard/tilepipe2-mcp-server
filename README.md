# tilepipe2-mcp-server

Model Context Protocol server for TilePipe2 autotile generation workflows.

TilePipe2 source changes and headless automation hooks live in:

- https://github.com/Cheesewizard/TilePipe2

This repository contains the MCP protocol surface, tool definitions, workflow orchestration, and integration code that drives the TilePipe2 fork through configured headless commands.
